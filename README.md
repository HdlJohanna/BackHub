# BackHub
Simple Backup-Program for Windows

# Installation

You can either:  

  * Install via Forge:  
    `forge install backhub`  
  * Directly [Download the file](https://ftp.hdljohanna.repl.co/e/0/backhub.exe)   
  * Install via graw:
    `graw -Gxn https://ftp.hdljohanna.repl.co/e/0/backhub.exe`  
  

and just run it.
